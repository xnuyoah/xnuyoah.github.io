类型转换（Type Casting）

使用内置函数 type() 查看数据类型，数据类型转换是将一种数据类型转换为另一种数据类型的过程

  • int()：将数据转换为整数类型, float 转 int 会丢失精度
  • float()：将数据转换为浮点数类型
  • str()：将数据转换为字符串类型
  • bool()：将数据转换为布尔类型
  • list()：将数据转换为列表
  • tuple()：将数据转换为元组
  • set()：将数据转换为集合
  • dict()：将符合字典结构的数据转换为字典

# 转换为整数
x = int("10")     # x = 10

# 查看数据类型
print(type(x))  # <class 'int'>

# 转换为浮点数
y = float("3.14") # y = 3.14

# 转换为字符串
z = str(100)      # z = "100"

# 转换为布尔值
a = bool(1)       # a = True

# 转换为列表
b = list("hello") # b = ['h', 'e', 'l', 'l', 'o']

# 转换为元组
c = tuple([1, 2, 3]) # c = (1, 2, 3)

# 转换为集合
d = set([1, 2, 2, 3]) # d = {1, 2, 3}

# 转换为字典
e = dict([('name', 'xxx'), ('age', 18)]) # e = {'name': 'xxx', 'age': 18}

运算符（Operators）

Python 中运算符分为多种类型，用于执行不同的操作。常见的运算符包括算术运算符、比较运算符、逻辑运算符和位运算符。

1) 算术运算符

  • +：加法
  • -：减法
  • *：乘法
  • /：除法
  • //：整除
  • %：取余
  • **：幂运算

x = 10
y = 3
print(x + y)  # 13
print(x - y)  # 7
print(x * y)  # 30
print(x / y)  # 3.333...
print(x // y) # 3
print(x % y)  # 1
print(x ** y) # 1000

2) 比较运算符

  • ==：等于
  • !=：不等于
  • > ：大于
  • <：小于
  • >=：大于等于
  • <=：小于等于

x = 5
y = 10
print(x == y)  # False
print(x != y)  # True
print(x > y)   # False
print(x < y)   # True
print(x >= 5)  # True
print(x <= y)  # True

3) 逻辑运算符

  • and：当两个条件都为真时，结果为 True
  • or：只要一个条件为真，结果为 True
  • not：取反运算，布尔值 True 变 False，反之亦然

a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

4) 位运算符

  • &：按位与
  • |：按位或
  • ^：按位异或
  • ~：按位取反
  • <<：左移
  • >>：右移

x = 5  # 5 的二进制为 101
y = 3  # 3 的二进制为 011
print(x & y)  # 1，二进制为 001
print(x | y)  # 7，二进制为 111
print(x ^ y)  # 6，二进制为 110
print(~x)     # -6，取反操作
print(x << 1) # 10，左移一位
print(x >> 1) # 2，右移一位

5) 赋值运算符

  • =：直接赋值
  • +=：加法赋值
  • -=：减法赋值
  • *=：乘法赋值
  • /=：除法赋值
  • %=：取余赋值
  • //=：整除赋值
  • **=：幂赋值
  • &=：按位与赋值
  • |=：按位或赋值
  • ^=：按位异或赋值
  • <<=：左移赋值
  • >>=：右移赋值

x = 10
x += 5   # 等同于 x = x + 5
print(x) # 15
