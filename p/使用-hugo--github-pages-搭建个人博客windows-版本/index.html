使用 Hugo + Github Pages 搭建个人博客(windows 版本)

Aug 25, 2024
阅读时长: 2 分钟

Hugo 下载

前往 Hugo Tags 选择 hugo_extended_xxx_windows-amd64.zip 版本下载后并解压

Hugo 解压安装

win + s 搜索 环境变量

点击 编辑环境变量，依次打开 环境变量 > 系统变量 > Path > 新建 并将 Hugo 解压路径添加进去

查看 hugo 版本 

1
2
3

hugo version

# 安装成功输出示例: hugo xxx

搭建个人博客

创建博客

  • 打开 cmd 窗口
1

hugo new site blogs
  • 本地预览
1
2
3

hugo server -D

# 访问地址: http://localhost:1313/

配置主题

配置主题时对主题进行修改可能会产生冲突，可以先 fork 主题仓库到自己的账户，使用 git submodule 对修改后的主题进行维护

  • 使用 git submodule
1
2
3

cd blogs
git init
git submodule https://github.com/xnuyoah/hugo-theme-stack themes/hugo-theme-stack
  • 复制 exampleSite 目录到主目录下然后修改配置
1

cp -rf themes/hugo-theme-stack/exampleSite/* ./

name: hugo deploy

on:
  push:
    branches:
      - main  # 设置部署分支

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
        - name: Checkout
          uses: actions/checkout@v4
          with:
              submodules: true  # 获取 hugo themes
              fetch-depth: 0

        - name: Setup Hugo
          uses: peaceiris/actions-hugo@v3
          with:
              hugo-version: "0.133.0"  # hugo 版本
              extended: true  # 启用 extended

        - name: Build Web
          run: hugo -D

        - name: Deploy Web
          uses: peaceiris/actions-gh-pages@v4
          with:
              PERSONAL_TOKEN: ${{ secrets.token变量名 }}
              EXTERNAL_REPOSITORY: github名/仓库名
              PUBLISH_BRANCH: main
              PUBLISH_DIR: ./public
              commit_message: auto deploy
